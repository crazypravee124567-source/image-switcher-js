# 🏏 Image Switcher — Cricket Legends

A beginner JavaScript project that switches between two player images using DOM manipulation — `getElementById`, `image.src`, and `if/else`.

## 🚀 Features

- 🔄 Switch between two cricket player cards with one button
- 🎞️ Smooth fade animation on image switch
- 🏷️ Player name, role, and jersey number shown on image
- ⚪ Dot indicators showing which player is active

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

---

## 🧠 JavaScript Concepts Used

| Concept | Usage |
|---|---|
| `getElementById()` | Select the image element |
| `image.src` | Change the image by updating its source URL |
| `if / else` | Toggle between player 0 and player 1 |
| `classList` | Add/remove fade and active dot classes |

## 📁 Project Structure

```
image-switcher/
├── index.html       # Main file (HTML + CSS + JS all in one)
├── virat.jpg        # Virat Kohli image (save locally)
├── sachin.jpg       # Sachin Tendulkar image (save locally)
└── README.md        # Project documentation
```

> 💡 **Tip:** Download the player images and save them locally as `virat.jpg` and `sachin.jpg`. Then update the `src` in the code to `"virat.jpg"` and `"sachin.jpg"` so the images always load — even without internet.

## ⚙️ How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/image-switcher.git
   ```
2. Add your player images (`virat.jpg`, `sachin.jpg`) to the folder
3. Open `index.html` in your browser

No installs needed!

## 🌐 How to Deploy on GitHub Pages

2. Click **Pages** in the left sidebar
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
